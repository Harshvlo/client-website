# VEXARO Courier Solutions — Official Website v1.0

A modern, premium logistics website built for VEXARO Courier Solutions Pvt. Ltd. (2026 Edition)

## Quick Start

```bash
# Install dependencies
npm install

# Start development server (opens at http://localhost:5173)
npm run dev

# Production build
npm run build && npm run preview
```

## Pages Included

| Route | Page |
|-------|------|
| `/` | Home — Hero, Stats, Services, Process, Testimonials, CTA |
| `/about` | About — Story, Mission/Vision/Values, Leadership, Network |
| `/services` | Services — All 8 service categories with benefits & CTAs |
| `/track` | Track Shipment — AWB search with status timeline (mock data) |
| `/contact` | Contact — Validated form, address, social links, map |
| `/faq` | FAQ — 15+ questions, accordion, category filter tabs |
| `/careers` | Careers — Job listings with apply flow, culture & perks |
